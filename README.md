# James Bell - Personal Website

A personal website for James Bell, a theatre maker, writer, business manager and photographer from Auckland, New Zealand.

## Features

- Responsive design that works on desktop, tablet, and mobile
- Hero section with portrait image
- About section with biography
- Links to blog, photos, LinkedIn, and other projects
- Clean, modern design with custom color scheme

## Technologies

- HTML5
- CSS3 with CSS Grid and Flexbox
- Google Fonts (Poppins and Lora)

## Local Development

Simply open `index.html` in your web browser to view the site.

## Customization

To customize the content:

1. Edit the hero greeting in the `<h1>` tag (line 215)
2. Update the callout and bio text in the about section (lines 227-238)
3. Modify the links in the links section (lines 247-252)
4. Replace the image files with your own photos
